1. 준비: 컴포넌트 & 스케치

모든 3D 모델링은 '어떤 부품을 만들 것인가'를 정의하는 것에서 시작합니다. 그 다음, 3D의 바탕이 되는 2D 설계도, 즉 '스케치'를 그립니다.

1-1. 새 컴포넌트 만들기

모델링 작업을 시작하기 전, 독립된 부품 서랍인 '컴포넌트'를 만드는 습관이 중요합니다. 이렇게 해야 작업이 엉키지 않고 깔끔하게 관리됩니다.

`Assemble` > `New Component` > `OK`

1-2. 트레이 밑그림 그리기

3D의 기본 설계도인 2D 스케치를 그립니다. 바닥 평면(XY Plane)에 중심점을 기준으로 사각형을 그리고, 정확한 치수를 입력합니다.

  • `Create Sketch` 클릭 후, 바닥 평면(빨강-초록 축 사이) 선택
  • `Create > Rectangle > Center Rectangle`로 원점에서 사각형 그리기
  • `Sketch Dimension` (단축키 `D`)으로 가로 100mm, 세로 50mm 치수 입력
  • `Finish Sketch` 클릭하여 스케치 마무리
⭐ 핵심 팁: 치수 기입 후 스케치 선이 모두 검은색으로 변하면 '완전 구속' 상태입니다. 이는 수정과 관리가 용이한 좋은 스케치라는 의미입니다.

2. 입체화: 돌출 (Extrude)

2D 스케치에 두께를 부여하여 3D 입체 형상으로 만드는 가장 기본적이고 강력한 기능입니다.

트레이 몸체 만들기 (영상 07:00 ~ 08:30)

완성된 사각형 스케치 면을 위로 잡아당겨 3차원 박스 형태로 만듭니다.

  • 스케치 면을 클릭하여 선택 (파란색으로 변함)
  • `Extrude` (단축키 `E`) 실행
  • 높이(Distance) 값으로 10mm 입력 후 `OK`
💡 알아두기: Extrude 기능은 단순히 입체를 만드는 것(Join) 외에도, 파내거나(Cut), 교차하는 부분만 남기는(Intersect) 등 다양하게 활용됩니다.

3. 속 파내기: Shell vs Extrude Cut

트레이의 내부를 비워내 물건을 담을 공간을 만듭니다. 이 작업에는 두 가지 대표적인 방법이 있습니다. 아래 탭을 클릭하여 각 방법의 차이점을 비교해 보세요.

더 쉽고 빠른 'Shell' 기능

모델의 특정 면을 터서, 마치 조개껍데기처럼 일정한 두께를 남기고 속을 파내는 가장 효율적인 방법입니다.

  • `Modify > Shell` 실행
  • 트레이의 윗면을 클릭하여 제거할 면으로 선택
  • 벽 두께(Thickness)를 2mm로 입력하면 완성!

장점: 매우 빠르고 간편하며, 복잡한 형태에도 잘 적용됩니다.

스케치로 파내는 'Extrude Cut'

파낼 모양을 직접 스케치한 뒤, Extrude 기능을 'Cut' 모드로 사용하여 파내는 정석적인 방법입니다.

  • 트레이 윗면에 `Create Sketch`
  • `Offset` (단축키 `O`)으로 바깥 테두리에서 안쪽으로 -2mm 간격의 사각형 생성
  • `Finish Sketch` 후, 안쪽 사각형 면을 선택하고 `Extrude` (단축키 `E`) 실행
  • 아래쪽으로 -8mm 만큼 파내기 (Distance = -8mm)

장점: 파내는 깊이나 모양을 자유롭게 조절할 수 있습니다.

4. 다듬기: 필렛 (Fillet)

날카로운 모서리를 부드럽게 둥글려주는 기능입니다. 모델의 완성도를 높이고 실제 제품처럼 보이게 하는 중요한 마감 작업입니다.

모서리 둥글게 깎기 (영상 14:00 ~ 20:00)

트레이의 안쪽과 바깥쪽 모서리를 각각 다른 값으로 둥글게 처리하여 자연스러운 형태를 만듭니다.

  • `Modify > Fillet` (단축키 `F`) 실행
  • 바깥쪽 바닥 모서리 선 선택 후, 반지름(Radius) 5mm 적용
  • 다시 `Fillet` 실행, 안쪽 바닥 모서리 선 선택 후, 반지름 3mm 적용
  • 마지막으로 윗면 테두리(안/밖) 선택 후, 반지름 1mm 적용하여 마무리
⭐ 핵심 팁: Fillet 작업 시, 값이 큰 모서리부터 작은 모서리 순서로 진행하면 오류 없이 안정적으로 작업할 수 있습니다.

5. 응용: 칸막이 추가

지금까지 배운 기능들을 모두 활용하여 트레이 내부에 칸막이를 만들어 봅시다. 새로운 기능 없이, 배운 것만으로도 충분히 응용이 가능합니다.

배운 기능 총동원하기 (영상 20:00 ~ 끝)

트레이 안쪽 바닥에 새로운 스케치를 그리고, 돌출시켜 기존 몸체와 합치는 방식으로 칸막이를 만듭니다.

  • 트레이 안쪽 바닥면에 `Create Sketch`
  • `Center Rectangle`로 칸막이 스케치 (예: 10mm x 46mm)
  • `Finish Sketch` 후, 칸막이 스케치를 `Extrude` (단축키 `E`)
  • 돌출 높이는 트레이 윗면까지, Operation은 **'Join'**으로 설정하여 기존 몸체와 합치기
  • 칸막이 모서리도 `Fillet` (단축키 `F`)으로 부드럽게 다듬어주면 완성!
